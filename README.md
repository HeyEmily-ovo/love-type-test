# 💘 恋爱类型测试

30 道题，发现你在爱情里的真实模样。

基于依恋理论（ECR）设计，从 6 个维度分析你的恋爱模式，输出 8 种恋爱类型 + 雷达图 + 详细建议。

## 快速开始

```bash
npx serve .          # 或 python -m http.server 8080
```

纯静态网站，无需安装依赖。

## 8 种恋爱类型

| 类型 | 一句话描述 |
|------|-----------|
| 🏠 安全感满格型 | 爱得从容，处得舒服 |
| 💗 恋爱脑上头型 | 爱到深处自然"黏" |
| 🫧 回避型依恋 | 靠近会窒息，离开会想念 |
| 🎭 纠结矛盾型 | 又想要又害怕 |
| 👑 强占有欲型 | 醋王本王驾到 |
| 🌸 甜宠浪漫型 | 把恋爱过成偶像剧 |
| ✨ 独立独美型 | 我自盛开，蝴蝶爱来不来 |
| 🧘 理性冷静型 | 佛系恋爱代言人 |

## 6 个分析维度

依恋焦虑 · 依恋回避 · 占有欲 · 浪漫表达 · 独立性 · 信任感

## 功能

- 30 题 Likert 量表，选择后自动跳转
- ECharts 雷达图 + 柱状图可视化
- 逐维度高低分解读 + 个性化建议
- 最佳匹配推荐
- 历史记录：自动保存最近 20 次测试结果，支持查看和删除
- 恋爱图鉴：8种类型翻转卡片 + 8×8 配对兼容矩阵
- 响应式适配桌面/移动端
- 纯前端，数据存 localStorage，无需后端

## 技术栈

HTML5 · CSS3 · 原生 JavaScript · [ECharts 5.5](https://echarts.apache.org/)
